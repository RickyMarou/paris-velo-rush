# Paris Velo Rush

A comical pseudo-3D arcade road racer set in Paris. Hop on a Velib and ride through the streets of Paris as fast as you can.

**[Play now](https://rickymarou.github.io/paris-velo-rush/)**

## Controls

- **Arrow Up** — Accelerate
- **Arrow Down** — Brake
- **Arrow Left / Right** — Steer
- **Enter** — Start game
- **Touch** — Left/right thirds to steer, touch to accelerate
